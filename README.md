# 📊 Superstore Sales Analysis

## 📌 Project Overview

The **Superstore Sales Analysis** project is an end-to-end data analytics capstone project that demonstrates the complete analytics lifecycle—from data cleaning and SQL analysis to exploratory data analysis (EDA) and interactive dashboard development. The objective is to transform raw sales data into meaningful business insights that support data-driven decision-making.

---

## 🎯 Project Objectives

* Clean and prepare raw sales data for analysis.
* Analyze sales performance, profitability, customer behavior, and regional trends.
* Answer key business questions using SQL queries.
* Perform Exploratory Data Analysis (EDA) using Python.
* Build an interactive Power BI dashboard for business users.
* Provide actionable recommendations based on data insights.

## 🔄 Project Workflow

### 1. Data Cleaning (Excel)

* Cleaned missing and inconsistent values.
* Performed data transformation using Power Query.
* Prepared the dataset for further analysis.

### 2. SQL Analysis

* Wrote SQL queries to answer business questions.
* Analyzed sales, profit, orders, products, customers, and regional performance.
* Extracted insights using filtering, grouping, aggregation, joins, and sorting.

### 3. Exploratory Data Analysis (Python)

* Imported and analyzed the dataset using Pandas.
* Performed descriptive statistical analysis.
* Identified sales trends and business patterns.
* Created visualizations using Matplotlib.

### 4. Power BI Dashboard

* Built an interactive dashboard with KPIs and charts.
* Created DAX measures for business metrics.
* Added slicers and filters for interactive analysis.

### 5. Business Insights

* Identified top-performing regions and categories.
* Analyzed monthly sales trends.
* Compared sales and profit across categories and sub-categories.
* Highlighted products contributing most to revenue and profit.
* Recommended areas for improving sales performance and profitability.
